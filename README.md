# Modern C++ Utility Module

An enterprise-grade, cross-platform C++ utility template demonstrating modern software engineering best practices. 

Currently implemented with a lightweight **BMI Calculator Module** to validate the automated CI/CD pipeline and cross-platform CMake build system.

## 🚀 Key Engineering Features

- **Cross-Platform Compatibility:** Successfully compiles on Linux (Ubuntu), macOS, and Windows.
- **Modern Build System:** Fully configured using `CMake` for dependency management and build generation.
- **Automated CI/CD Pipeline:** Integrated with **GitHub Actions** for automated building and status checks upon every commit.
- **Clean Architecture:** Strict separation of source files (`src/`) and build configurations.

## 🛠️ Tech Stack

- **Language:** Modern C++ 
- **Build Tool:** CMake
- **CI/CD:** GitHub Actions
- **Version Control:** Git

## ⚙️ How to Build and Run (Local)

If you wish to build this project locally, ensure you have CMake and a C++ compiler installed, then run:

cd build
